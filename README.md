# Focus Hub: Sistema IoT de Productividad Sincronizada
Focus Hub es un ecosistema de productividad basado en el concepto de "Gemelo
Digital". Combina una aplicación móvil moderna para la gestión de tareas con un
dispositivo físico de escritorio (Hardware) que permite al usuario enfocarse en su
trabajo mediante la técnica Pomodoro, minimizando las distracciones digitales.
El sistema garantiza una sincronización bidireccional casi instantánea entre el mundo
físico y el digital utilizando la nube de AWS como orquestador central.
